1. pip install -r requirements.txt
不行的話就把requirements.txt裡的所有套件全部安裝一遍

2. 執行python -m streamlit run app.py
會有Local URL: 
開啟那個URL就會有網頁了

3. 使用說明

左邊的側邊欄可以設定環境

(1)Faster whisper模型:base、small、medium(我是用medium，雖然時間比較長但比較準確，還是以準確度優先)

(2)可以選擇要用CPU或GPU，沒有GPU預設為CPU

(3)Prompt:可以輸入一些關於此會議的關鍵字或語句，使模型更加準確

方法一：網頁錄音

操作步驟：
點選「音訊輸入」
選擇「網頁錄音」
按下「開始錄音」
說話
按下「停止錄音」
切換至「逐字稿」
點擊「開始轉錄」

方法二：上傳音檔

支援格式：wav、mp3、m4a

操作步驟：
點選「音訊輸入」
選擇「上傳音檔」
選擇音檔
切換至「逐字稿」
點擊「開始轉錄」

有留摘要和RAG查詢的地方，如果覺得網頁太醜的話再討論看看要怎麼改
